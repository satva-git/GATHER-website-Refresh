## How to Use

### View the Features
1. Open the website with `?review=token` parameter
2. Right-click to add comments
3. Use Tab/Arrows to navigate with keyboard
4. Type @ to mention team members
5. Click to add emoji reactions
6. Pin critical comments
7. View edit history on changes

### Keyboard Shortcuts
- **Tab**: Navigate elements
- **Arrows**: Move through comments
- **Enter**: Open focused comment
- **Escape**: Close modals
- **Ctrl+Enter**: Submit reply

### Backend Integration Ready
- POST `/api/comments` - Create comment
- PATCH `/api/comments/:id` - Edit/update
- DELETE `/api/comments/:id` - Delete
- POST `/api/comments/:id/reactions` - Add reaction
- POST `/api/comments/:id/pin` - Pin toggle
- GET `/api/comments/:id/history` - Edit history
